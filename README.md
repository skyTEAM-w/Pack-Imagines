# Pack-Imagines

这是一个基于PaddleOCR与openpyxl的，应用于打包打卡数据并进行自动检查的程序。

## 使用

### 配置文件

与`pack_img.py`同目录下必须包含以下文件：

1. 当日从腾讯文档导出的`.xlsx`文件
2. 人员名单`list.txt`, 每个姓名独占一行

开始运行前，请保证使用的Python解释器已经安装`PaddleOCR`与`openpyxl`有关库，第一次使用`PaddleOCR`时需要下载数据库，请等待。

### 运行

按照控制台输出的提示输入即可，程序会自动从网络中下载所有图片并保存的指定的文件夹内，之后可以选择是否对图片进行检查。
检查时间长短由电脑配置决定。
